# Git_Svn_DS_exp_gui
Git、SVN和DS_Store三种常见源码泄露Dump 界面化工具
一个强大的网站源码泄露利用工具，支持多种泄露类型检测与利用
Git exp
SVN exp
DS_Store exp


## 功能特点

- **全面的泄露检测**：支持Git、SVN和DS_Store三种常见源码泄露利用
- **多线程扫描**：高效并发处理，加速扫描过程
- **文件浏览与预览**：内置文件浏览器和内容查看器
- **代理支持**：灵活的网络代理配置选项
- **详细的日志记录**：完整记录扫描过程和结果
